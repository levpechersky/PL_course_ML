Summary:
* The tests test the functions exit_maze, one_path_maze.
* ex3test_invalid.sml tests all, valid and invalid, 1x1, 1x2, 2x2 mazes.
* ex3test_valid.sml tests all valid 1x1, 1x2, 2x2, 2x3 mazes.

Usage:
* Create the ex3.sml file with the exit_maze, one_path_maze 
  functions in the same folder with the tests.
* Open console.
* Run: sml ex3test_invalid.sml > out_invalid
* Run: sml ex3test_valid.sml > out_valid
* Compare with our output.
